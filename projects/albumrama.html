Albumrama

A 10 image collage maker revolving around generating album covers using the Last.fm API

Albumrama is a single-page site that creates a 10 image collage using the Last.fm API. This project was originally a submission for a final project however I have committed to updating it consistently. An update log can be found at the bottom of the site. My inspiration for this project stemmed from the original Topsters 2 website. When approaching this project, I wanted to create something visually appealing, and very straightforward. While the functionalities of the Topsters 2 website were nice, I couldn't help but be put off by it's design. This project makes use of HTML, CSS, jQuery, Express, Node.js, and is run with Heroku.

The first iteration of this project allows you to search for an album name, an artist name, or if you know both, an album name and an artist name. The Last.fm API returns data based on your search and I had to double check which results had an album image to display, otherwise they wouldn't.

The user selects the albums they'd like to add to their collage by clicking. If for whatever reason they'd like to clear their entire grid, the trash can icon resets all entries. For individual album removal, the user has to click on the album they want to remove within their collage.

The color scheme I chose was generated with the help of coolors.co. I was aiming for a warm color scheme. As stated before, this project is continuously updated. Check it out here!
